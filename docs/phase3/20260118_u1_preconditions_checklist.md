# Phase 3-C / Epic U-1 전제 조건 검증 체크리스트
최초작성일자: 2026-01-18
최종수정일자: 2026-01-18

## 목적

U-1 사용자 기능(읽기 전용) 공개 전에 C-1~C-4 준비 상태를 확인하는 체크리스트다.

---

## 1. C-1 운영 안정성

- [ ] 배치 실행/재처리 흐름 정상 동작 확인
- [ ] 운영 알림 발송 및 확인 처리 확인
- [ ] 감사 로그 생성 및 조회 확인
- [ ] 수동 중단/재개 정책 문서화
- **증빙**: 통합 테스트 결과, 감사 로그 샘플

---

## 2. C-2 데이터 품질·계보·재현성

- [ ] 스냅샷/라인리지/검증 DDL 정합성 확인
- [ ] 데이터 출처 및 적재 이력 추적 가능
- [ ] 재현성 확보(동일 입력 결과 동일)
- [ ] 데이터 품질 규칙 문서화
- **증빙**: DDL 문서, 품질 규칙 정의서

---

## 3. C-3 성과 분석 고도화

- [ ] 성과 산식/누적/연환산 정의 완료
- [ ] 벤치마크/리스크 지표 계산 기준 확정
- [ ] 결과 버전 활성화 단일성 확인
- **증빙**: 성과 계산 엔진 상세 설계서

---

## 4. C-4 관리자 통제

- [ ] RBAC 적용 및 권한 매트릭스 일치
- [ ] 승인/정정/감사로그 흐름 검증
- [ ] 관리자 API와 사용자 API 분리
- **증빙**: C-4 검증 체크리스트 결과, 감사 로그 기록

---

## 5. 사용자 노출 승인

- [ ] C-1~C-4 체크리스트 완료 확인
- [ ] 사용자 문구(투자 권유 금지) 최종 점검
- [ ] Go-Live 담당 승인 기록
- **증빙**: 승인 로그 또는 서명 기록
