# Phase 8-B 결과 고지 문구 v3

본 문서는 입력 다양성 확장(Phase 8-B)에 따라 추가되는 고지 문구를 정의한다.
모든 문구는 정보 제공 목적에 한정하며, 추천/판단/우열 표현을 포함하지 않는다.

## 1. 기본 고지(공통)
- 본 결과는 사용자가 직접 선택한 구성에 대해 과거 데이터로 계산된 분석 결과입니다.
- 본 서비스는 투자 자문 또는 투자 권유를 제공하지 않으며, 미래 성과를 예측하거나 보장하지 않습니다.

## 2. 통화 기준 고지
- 본 결과는 KRW 기준으로 계산되었습니다.
- 본 결과는 USD 기준으로 계산되었습니다.

## 3. 수익 기준 고지
- 본 결과는 가격 수익(Price Return) 기준으로 계산되었습니다.
- 본 결과는 배당을 포함한 총수익(Total Return) 기준으로 계산되었습니다.

## 4. 데이터 범위 고지
- 선택한 기간의 과거 데이터가 제한되어 일부 계산이 생략될 수 있습니다.

## 5. 적용 위치
- 결과 요약 화면
- 상세 결과 화면
- 비교 화면
- 저장/내보내기 화면

## 6. 금지 표현
- 추천/최적/우수/효율/승자/랭킹/점수 관련 표현
- 미래 예측 또는 기대 수익 관련 표현
