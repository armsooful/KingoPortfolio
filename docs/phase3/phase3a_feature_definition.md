
# Phase 3-A 기능 정의서  
## 설명·안심 중심 포트폴리오 해석 서비스

## 1. Phase 3-A 개요
Phase 3-A의 목적은 일반 사용자가 자신의 포트폴리오 결과를 이해하고, 불안을 줄이며, 스스로 판단할 수 있도록 돕는 것이다.

- 종목 추천 ❌
- 투자 판단 유도 ❌
- 결과 해석 및 맥락 제공 ⭕️

## 2. 핵심 컨셉
정답을 주지 않는다. 대신 이해를 준다.

## 3. 기능 범위
### In Scope
- 성과 지표 해석
- 비교 맥락 설명
- 설명형 리포트 UI

### Out of Scope
- 종목 추천
- 매수/매도 시점 제시
- 실거래 연계

## 4. 주요 기능
### 4.1 성과 해석 엔진
입력: 수익률, 변동성, MDD, 기간  
출력: 자연어 설명 문구

### 4.2 맥락 비교
시장 지수 및 기준 포트폴리오와의 성격 비교

### 4.3 UI 구성
요약 카드 → 성과 해석 → 위험 구간 → 비교 설명

## 5. 규제 기준
투자 권유 표현 금지, 과거 데이터 기반 고지 필수
