# Phase 3-D / 피드백 문구 템플릿 + 금지 가드 세트
최초작성일자: 2026-01-19
최종수정일자: 2026-01-19

## 1. 피드백 문구 원칙
- 비교 금지 (우수/열위, 상위/하위 등)
- 평가 금지 (좋음/나쁨, 훌륭함 등)
- 예측 금지 (앞으로/예상/전망 등)
- 사실 서술형만 허용

## 2. 템플릿 세트

### 2.1 재시도
- "최근 7일 동안 이 단계에서 평균 {retry_count}회 재시도가 발생했습니다."
- "지난주 대비 재시도 건수가 {delta}건 {direction}했습니다."

### 2.2 완료율
- "최근 7일 동안 이 단계의 완료율은 {completion_rate}%였습니다."
- "지난 7일 평균 완료까지 {avg_duration_min}분이 소요되었습니다."

### 2.3 이탈 지점
- "최근 7일 동안 이 단계에서 이탈이 가장 많이 발생한 지점은 {dropoff_point}입니다."
- "최근 7일 동안 {dropoff_point}에서 {dropoff_count}건의 이탈이 발생했습니다."

### 2.4 상태 변화
- "최근 7일 동안 이 단계의 상태 전이 횟수는 {transition_count}회입니다."
- "최근 7일 동안 보류 상태가 {deferred_count}회 기록되었습니다."

## 3. 금지 가드 세트

### 3.1 금지 키워드(예시)
- 추천, 권유, 사세요, 파세요
- 보장, 확실한 수익, 손실 없음, 무위험
- 예상, 전망, 미래, 예측, 확신

### 3.2 금지 패턴
- "~이 더 좋습니다"
- "~이 우수합니다"
- "~을 추천합니다"

## 4. 적용 규칙
- 템플릿 렌더링 후 금지어 검사 수행
- 위반 시 기본 안내 문구로 대체

## 5. 기본 대체 문구
- "현재 기록된 사실을 요약하여 제공합니다."
- "최근 기록된 상태 변화만을 안내합니다."
