# Phase 3-B 가격·상품 구조 정의서
## 프리미엄 리포트 기반 수익화 모델

---

## 1. 문서 목적

본 문서는 Phase 3-B 단계에서 제공되는 프리미엄 리포트 상품의 가격 정책과 상품 구조를 정의한다.
본 정의서는 개발, 기획, 마케팅, 법무 검토의 공통 기준 문서로 사용한다.

**⚠️ 본 서비스는 투자 자문, 추천, 매매 유도를 제공하지 않습니다.**

---

## 2. 기본 원칙

1. **추천 없이 과금한다** - 종목 추천이나 매매 조언 없이 해석 서비스만 제공
2. **판단이 아닌 설명에 대해 비용을 받는다** - 투자 판단 유도 없음
3. **무료 기능만으로도 서비스의 핵심 가치는 전달된다** - 기본 해석 무료 제공
4. **유료 기능은 저장·기록·심화 설명에 한정한다** - 부가 기능에 과금
5. **모든 상품은 규제 리스크를 최소화한다** - 자본시장법 준수

---

## 3. 상품 구성 개요

### 3.1 상품 유형

| 유형 | 설명 | 비용 |
|------|------|------|
| 기본 해석 리포트 | 웹 화면 조회 | 무료 |
| 프리미엄 리포트 (단건) | PDF 다운로드 | 유료 |
| 프리미엄 구독 | 월정액 서비스 | 유료 |

---

## 4. 무료 상품 정의 (Basic Tier)

### 4.1 제공 기능

| 기능 | 내용 |
|------|------|
| 포트폴리오 성과 요약 | 핵심 지표 간단 해석 |
| 웹 화면 조회 | 온라인 조회 전용 |
| 월 분석 횟수 | 3회 |
| 히스토리 저장 | 최근 5건 |
| 벤치마크 비교 | 1개 (KOSPI) |

### 4.2 제한 사항

- PDF 다운로드: 기본 PDF만 (1페이지)
- 기간별 비교: 미제공
- 프리미엄 리포트: 미제공

---

## 5. 유료 상품 – 프리미엄 리포트 (Standard Tier)

### 5.1 제공 기능

| 기능 | 내용 |
|------|------|
| PDF 리포트 다운로드 | 기본 + 프리미엄 PDF (5-7페이지) |
| 지표별 상세 설명 | CAGR, 변동성, MDD, 샤프비율 |
| 위험 구간 분석 | MDD 발생 구간, 회복 과정 |
| 벤치마크 비교 | 5개 (KOSPI, KOSDAQ, S&P500 등) |
| 월 분석 횟수 | 30회 |
| 히스토리 저장 | 최근 50건 |
| 기간별 비교 | 최대 4개 기간 |

### 5.2 가격

| 구분 | 가격 | 비고 |
|------|------|------|
| 월 구독 | ₩9,900 | - |
| 연 구독 | ₩99,000 | 17% 할인 |

## 10. 규제 준수 기준

### 10.1 투자자문업 규제 회피

본 서비스는 **자본시장법상 투자자문업에 해당하지 않음**을 명확히 합니다.

| 투자자문업 요소 | 본 서비스 대응 |
|-----------------|----------------|
| 특정 종목 추천 | ❌ 미제공 |
| 매매 시점 조언 | ❌ 미제공 |
| 수익률 보장 | ❌ 금지 |
| 개인별 맞춤 조언 | ❌ 일반적 해석만 제공 |

### 10.2 필수 고지 문구

모든 서비스 화면 및 리포트에 표시:

```
⚠️ 본 서비스는 과거 데이터 기반 교육적 해석이며,
투자 권유나 자문이 아닙니다.
과거 성과가 미래 수익을 보장하지 않습니다.
모든 투자 판단의 책임은 본인에게 있습니다.
```

### 10.3 금지 표현

| 금지 표현 | 허용 대안 |
|-----------|-----------|
| "이 종목을 사세요" | (사용 금지) |
| "수익이 보장됩니다" | "과거 성과는 미래를 보장하지 않습니다" |
| "반드시 오릅니다" | "역사적으로 상승 추세를 보였습니다" |
| "지금이 매수 타이밍" | (사용 금지) |
