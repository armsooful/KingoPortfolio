
# A-1 설명 자동 생성 로직 설계서 (Rule / Score 기반)

## 1. 목적
본 문서는 프리미엄 리포트 및 설명형 UI에 사용되는 설명 문구를
성과 지표 기반으로 자동 생성하기 위한 로직을 정의한다.

본 로직은 투자 판단, 추천, 성과 평가를 수행하지 않는다.

---

## 2. 입력 지표
- CAGR
- 변동성 (Volatility)
- 최대 낙폭 (MDD)
- 투자 기간
- 리밸런싱 여부

---

## 3. 상태 분류 규칙 (Rule)

### 3.1 CAGR 상태
| 조건 | 상태 |
|---|---|
| CAGR < 3% | 낮음 |
| 3% ≤ CAGR < 7% | 보통 |
| CAGR ≥ 7% | 높음 |

### 3.2 변동성 상태
| 조건 | 상태 |
|---|---|
| 하위 30% | 낮음 |
| 중간 | 보통 |
| 상위 30% | 높음 |

### 3.3 MDD 상태
| 조건 | 상태 |
|---|---|
| MDD > -10% | 안정 |
| -20% < MDD ≤ -10% | 주의 |
| MDD ≤ -20% | 큼 |

---

## 4. 스코어링 개념 (내부용)
- 안정성 점수 = 변동성 + MDD 가중 평균
- 성장성 점수 = CAGR 중심
※ 점수는 외부 노출 금지

---

## 5. 문구 매핑 로직
상태 조합에 따라 템플릿 라이브러리에서 문구 선택

예)
- CAGR: 보통 / 변동성: 낮음 → “완만하지만 안정적인 흐름”

---

## 6. 제어 파라미터
- 설명 길이: short / normal / long
- 톤: 중립 / 완화

---

## 7. 금지 사항
- 추천, 최적, 반드시 등 표현 사용 금지
- 미래 성과 암시 금지
