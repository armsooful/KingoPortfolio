# Phase 3-C / Epic U-1 오류·지연 처리 가이드
최초작성일자: 2026-01-18
최종수정일자: 2026-01-18

## 목적

사용자에게 데이터 오류나 지연이 발생했을 때 신뢰를 유지할 수 있는 안내 기준을 정의한다.

---

## 케이스별 처리

### 1) 데이터 미산출
- 안내 문구: "현재 최신 데이터가 준비되지 않았습니다. 잠시 후 다시 확인해주세요."
- 상태 표기: 참고용 배지 표시

### 2) 기준일 불일치
- 안내 문구: "표시된 기준일이 최신일과 다를 수 있습니다."
- 상태 표기: 경고 배지 표시

### 3) 시스템 오류
- 안내 문구: "일시적인 오류가 발생했습니다. 잠시 후 다시 시도해주세요."
- 기술 용어 노출 금지

---

## 공통 원칙

- 사용자 행동 유도 금지
- 해결 가능 시간/대안 제시
- 알림/경고 배지는 명확한 색상으로 구분
