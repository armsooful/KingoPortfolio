# Phase 8-A 분석 지표 정의 (v3)

작성일: 2026-01-21  
범위: Phase 8-A (Analysis Depth Expansion)

---

## 1. 목적
Phase 7 입력을 그대로 유지한 상태에서, 더 많은 과거 사실을 정교하게 제공한다.  
판단·추천·우열 표현은 추가하지 않는다.

---

## 2. 고정 지표 목록 (v3)

### 2.1 롤링 수익률
- 3Y 롤링 수익률
- 5Y 롤링 수익률
- 출력: 기간별 계산 값(표/그래프)
- 주의: “우수/효율/안정적” 표현 금지

### 2.2 롤링 변동성
- 3Y 롤링 변동성
- 출력: 기간별 계산 값(표/그래프)
- 주의: “안정적/공격적” 표현 금지

### 2.3 연도별 성과 분해
- 연도 단위 성과 테이블
- 출력: 연도별 수익률 수치만 제공
- 주의: 자동 비교/결론 없음

### 2.4 기여도(Contribution)
- 종목/섹터별 수익 기여도
- 출력: 구성 요소별 기여 비중(수치)
- 금지: “원인/문제/책임” 표현

### 2.5 드로다운 구조 분석
- 최대 낙폭 구간 세분화
- 출력: 구간 정보(기간/낙폭 크기)
- 금지: 위험 경고 문구
