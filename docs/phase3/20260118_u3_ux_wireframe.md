# Phase 3-C / U-3 UX 와이어(초안)
최초작성일자: 2026-01-18
최종수정일자: 2026-01-18

## 1. 목적
U-3 고급 사용자 흐름을 위한 핵심 화면/플로우를 정의한다.

## 2. 화면 구성(요약)

### 2.1 사용자 설정 허브
- 목적: 프리셋/알림 설정 진입 허브
- 요소:
  - 프리셋 요약 카드(현재 기본 프리셋 표시)
  - 알림/노출 빈도 토글
  - 최근 활동 요약(최근 3건)

### 2.2 프리셋 관리 화면
- 목록 영역: 프리셋 리스트, 기본 프리셋 표시
- 액션: 생성/수정/삭제
- 필드: 이름, 타입(VIEW/SORT/FILTER), 설정 요약

### 2.3 알림/노출 빈도 설정
- 토글: 알림 사용 여부(옵트인)
- 빈도: LOW/STANDARD/HIGH
- 안내 문구: 과도한 알림 방지 가이드

### 2.4 활동/상태 히스토리
- 목록 영역: 이벤트 타임라인
- 상태 배지: IN_PROGRESS/DEFERRED/BLOCKED/COMPLETED
- 사유 표시: 차단/지연/시스템 오류 등

## 3. 핵심 플로우
- 설정 허브 -> 프리셋 관리 -> 기본 프리셋 설정
- 설정 허브 -> 알림 설정 변경
- 설정 허브 -> 활동 히스토리 확인

## 4. 상태/가드 UX
- 차단 시 메시지: 원인 + 대안 경로
- 지연 시: 예상 대기 시간 및 재시도 안내

## 5. 접근성 고려
- 키보드 탐색 가능
- 상태 배지 대비 기준 충족
- 스크린리더용 상태 텍스트 제공
