## 4. StockMatrix 지표의 실제 유용성 평가

### 강점 (진짜 유용한 것)
1. **복수 지표 합산 점수**: 개별 지표는 거짓 신호가 많지만, 여러 지표가 같은 방향을 가리키면 신뢰도가 크게 올라감. StockMatrix의 6개 카테고리 점수 체계는 이를 잘 구현함.
2. **ATR + 리스크/리워드**: 단순히 "매수/매도"가 아니라 "어디서 손절할지"까지 제시. 실전 매매에서 가장 중요한 부분.
3. **정배열/역배열 + ADX**: "지금 추세가 있는가?" → "있다면 방향은?" 이 두 질문에 답하는 조합.

### 약점 (과대 포장된 것)
1. **30개라는 숫자**: 마케팅 효과는 크지만, 실제로 CCI/Williams %R/ROC 등은 RSI/Stochastic/MACD와 정보가 중복됨. 독립적 신호를 주는 지표는 15~18개 정도.
2. **체결강도/매수매도 압력**: 일봉 데이터로는 정확한 계산이 불가능. 실시간 틱 데이터가 필요하여, 뉴스레터 형태에서는 제한적.
3. **추세선 분석**: 자동화가 매우 어려워서 AI가 실제로 정확한 추세선을 그리는지 의문.

### 핵심 인사이트
> StockMatrix의 진짜 가치는 "30개 지표" 자체가 아니라, **복수 지표를 종합 점수로 변환하여 비전문가도 한눈에 파악하게 한 UX**에 있다. 개별 지표 수를 늘리는 것보다 기존 지표를 점수 체계로 통합하는 것이 더 효과적이다.

## 5. Foresto Compass 접목 전략

### Phase A: 즉시 적용 가능 (기존 데이터로 구현, 1-2일)

**quant_analyzer.py에 6개 지표 추가:**

```
1. Stochastic Oscillator (%K, %D)
   - 입력: 고가, 저가, 종가 (pykrx OHLCV에서 확보 가능)
   - 공식: %K = (종가 - N일 최저가) / (N일 최고가 - N일 최저가) × 100

2. ATR (Average True Range)
   - 입력: 고가, 저가, 종가
   - 공식: TR = max(고가-저가, |고가-전일종가|, |저가-전일종가|)
   - ATR = TR의 14일 이동평균

3. ADX (Average Directional Index)
   - 입력: 고가, 저가, 종가
   - DI+, DI- 계산 → DX → ADX (14일 평균)

4. OBV (On Balance Volume)
   - 입력: 종가, 거래량
   - 종가 상승 시 +거래량, 하락 시 -거래량 누적

5. MA 정배열/역배열 판정
   - 기존 SMA 20/50/200 활용
   - 정배열: SMA20 > SMA50 > SMA200
   - 역배열: SMA20 < SMA50 < SMA200

6. 52주 고저 대비 위치
   - 입력: 252일 가격 데이터 (이미 보유)
   - 공식: (현재가 - 52주 최저) / (52주 최고 - 52주 최저) × 100
```

### Phase B: 종합 점수 체계 구축 (2-3일)

**StockMatrix 스타일 점수 카드:**

```
카테고리별 점수 (100점 만점):
├── 추세 점수 (가중치 25%)
│   ├── MA 정배열/역배열
│   ├── ADX 추세 강도
│   └── MACD 방향
├── 모멘텀 점수 (가중치 20%)
│   ├── RSI 위치
│   ├── Stochastic 위치
│   └── MACD 히스토그램 변화
├── 거래량 점수 (가중치 20%)
│   ├── 거래량 vs 평균거래량
│   └── OBV 추세
├── 변동성 점수 (가중치 15%)
│   ├── ATR 수준
│   ├── 볼린저밴드 %B
│   └── 밴드폭 변화
├── 가격 위치 점수 (가중치 10%)
│   ├── 52주 고저 대비
│   └── 지지/저항 근접도

    ├── Beta
    └── MDD

```

## 6. Foresto Compass의 차별적 우위

StockMatrix에 **없고** Foresto에만 있는 분석:

| 분석 영역 | Foresto 보유 | StockMatrix |
|----------|-------------|-------------|
| 재무제표 분석 (ROE, 마진, CAGR) | ✅ | ❌ |
| 밸류에이션 (PER/PBR/DCF/DDM) | ✅ | ❌ |
| 포트폴리오 구성/백테스트 | ✅ | ❌ |
| 리스크 지표 (Beta, Sharpe, Alpha) | ✅ | ❌ |
| 뉴스 감성 분석 (AI) | ✅ | ❌ |
| 투자 성향 진단 | ✅ | ❌ |
| 시나리오 시뮬레이션 | ✅ | ❌ |

> **결론**: StockMatrix는 기술적 분석 특화, Foresto Compass는 종합 투자 학습 플랫폼. 기술적 지표 6개를 추가하고 종합 점수 체계를 만들면, StockMatrix의 장점을 흡수하면서도 재무/밸류에이션/포트폴리오라는 고유 가치를 유지할 수 있다.
