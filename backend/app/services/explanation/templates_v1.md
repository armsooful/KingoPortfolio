# 설명 문구 템플릿 라이브러리 v1

본 문서는 버전 관리되며, 운영 변경은 PR + 리뷰를 통해서만 반영됩니다.

## 1. CAGR (연평균 수익률) 템플릿

### 1.1 상태별 설명 템플릿

| template_id | 상태 조건 | 설명 문구 |
|-------------|-----------|-----------|
| cagr_very_negative | cagr < -10% | 해당 기간 동안 연평균 {abs_cagr:.1f}%의 손실이 발생했습니다. 이는 초기 투자금 대비 상당한 가치 감소를 의미합니다. |
| cagr_negative | -10% <= cagr < 0% | 해당 기간 동안 연평균 {abs_cagr:.1f}%의 손실이 발생했습니다. 원금 대비 가치가 감소한 상태입니다. |
| cagr_low | 0% <= cagr < 3% | 연평균 {cagr:.1f}%의 수익률을 기록했습니다. 이는 일반적인 예금 금리 수준과 유사한 수익률입니다. |
| cagr_medium | 3% <= cagr < 7% | 연평균 {cagr:.1f}%의 수익률을 기록했습니다. 이는 과거 채권 수익률과 비슷한 수준입니다. |
| cagr_high | 7% <= cagr < 12% | 연평균 {cagr:.1f}%의 수익률을 기록했습니다. 이는 과거 주식시장 장기 평균 수익률과 유사한 수준입니다. |
| cagr_very_high | cagr >= 12% | 연평균 {cagr:.1f}%의 수익률을 기록했습니다. 이는 과거 시장 평균을 상회하는 수익률이나, 높은 수익률은 일반적으로 높은 위험을 동반합니다. |

### 1.2 맥락 설명

```
장기적으로 자산이 어떤 속도로 변화했는지를 보여주는 지표입니다.
단기 변동을 평균화한 결과이므로, 중간 과정의 굴곡은 반영되지 않습니다.
({period_years:.1f}년 기준, 과거 성과이며 미래 수익을 보장하지 않습니다.)
```

## 2. 변동성 (Volatility) 템플릿

### 2.1 상태별 설명 템플릿

| template_id | 상태 조건 | 설명 문구 |
|-------------|-----------|-----------|
| vol_very_low | vol < 5% | 연간 변동성이 {vol:.1f}%로, 가격 변동 폭이 작은 편입니다. 이는 예금이나 단기 채권과 유사한 안정성을 보입니다. |
| vol_low | 5% <= vol < 12% | 연간 변동성이 {vol:.1f}%로, 변동성이 낮은 가격 흐름을 보입니다. 채권형 자산과 유사한 변동성 수준입니다. |
| vol_medium | 12% <= vol < 20% | 연간 변동성이 {vol:.1f}%로, 일반적인 주식시장과 유사한 수준입니다. 단기적으로 상당한 가격 변동이 있을 수 있습니다. |
| vol_high | 20% <= vol < 30% | 연간 변동성이 {vol:.1f}%로, 상당히 높은 편입니다. 단기간에 큰 폭의 가격 변동을 경험할 수 있습니다. |
| vol_very_high | vol >= 30% | 연간 변동성이 {vol:.1f}%로, 매우 높은 수준입니다. 극심한 가격 변동이 발생할 수 있습니다. |

### 2.2 맥락 설명

```
가격이 오르내린 폭이 얼마나 컸는지를 나타냅니다.
변동성이 크다는 것은 수익 가능성과 함께 심리적 부담도 컸을 수 있음을 의미합니다.
```

## 3. MDD (최대 낙폭) 템플릿

### 3.1 상태별 설명 템플릿

| template_id | 상태 조건 | 설명 문구 |
|-------------|-----------|-----------|
| mdd_stable | mdd > -5% | 최대 낙폭이 {abs_mdd:.1f}%로, 하락폭이 제한적이었습니다. 상대적으로 변동이 적은 가치 흐름을 보였습니다. |
| mdd_caution_low | -10% < mdd <= -5% | 최대 낙폭이 {abs_mdd:.1f}%였습니다. 일시적인 조정 국면을 경험했으나 비교적 제한된 손실이었습니다. |
| mdd_caution | -20% < mdd <= -10% | 최대 낙폭이 {abs_mdd:.1f}%로, 상당한 하락을 경험했습니다. 이 정도 하락은 주식시장에서 종종 발생하는 조정 수준입니다. |
| mdd_large | -35% < mdd <= -20% | 최대 낙폭이 {abs_mdd:.1f}%로, 큰 폭의 하락을 경험했습니다. 고점 대비 1/3 이상의 가치 감소가 있었던 기간이 있었습니다. |
| mdd_severe | mdd <= -35% | 최대 낙폭이 {abs_mdd:.1f}%로, 극심한 하락을 경험했습니다. 이는 금융위기 수준의 하락폭으로, 회복에 상당한 시간이 필요했을 수 있습니다. |

### 3.2 맥락 설명

```
과거 기간 중 가장 크게 하락했던 구간을 의미합니다.
이 구간은 많은 투자자들이 가장 불안함을 느끼는 시점입니다.
```

### 3.3 회복 관련 추가 맥락

```
해당 낙폭은 {peak_date}부터 {trough_date} 사이에 발생했습니다.
이후 약 {recovery_months:.0f}개월 만에 고점을 회복했습니다.
분석 기간 내 고점을 회복하지 못했습니다.
```

## 4. 샤프 비율 (Sharpe Ratio) 템플릿

### 4.1 상태별 설명 템플릿

| template_id | 상태 조건 | 설명 문구 |
|-------------|-----------|-----------|
| sharpe_na | sharpe = None | 변동성이 0에 가까워 샤프 비율을 계산할 수 없습니다. |
| sharpe_negative | sharpe < 0 | 샤프 비율이 {sharpe:.2f}로, 위험 대비 수익이 부정적입니다. 무위험 수익률보다 낮은 수익을 기록했습니다. |
| sharpe_low | 0 <= sharpe < 0.5 | 샤프 비율이 {sharpe:.2f}로, 위험 대비 수익이 낮은 편입니다. 감수한 위험에 비해 초과 수익이 제한적이었습니다. |
| sharpe_medium | 0.5 <= sharpe < 1.0 | 샤프 비율이 {sharpe:.2f}로, 일반적인 수준입니다. 위험에 대한 적정 수준의 보상을 받았습니다. |
| sharpe_high | 1.0 <= sharpe < 2.0 | 샤프 비율이 {sharpe:.2f}로, 위험 대비 수익이 양호합니다. 감수한 위험에 비해 좋은 초과 수익을 기록했습니다. |
| sharpe_very_high | sharpe >= 2.0 | 샤프 비율이 {sharpe:.2f}로, 매우 높은 수준입니다. 그러나 이례적으로 높은 샤프 비율은 지속되기 어려울 수 있습니다. |

### 4.2 맥락 설명

```
샤프 비율은 (수익률 - 무위험수익률) / 변동성으로 계산됩니다.
무위험수익률 {rf_pct:.1f}%를 기준으로 계산되었습니다.
높을수록 위험 대비 효율적인 수익을 의미하지만, 과거 지표입니다.
```

## 5. 종합 요약 (Summary) 템플릿

### 5.1 수익률 설명

| template_id | 조건 | 문구 |
|-------------|------|------|
| summary_return_loss | cagr < 0 | 연평균 {abs_cagr:.1f}%의 손실 |
| summary_return_gain | cagr >= 0 | 연평균 {cagr:.1f}%의 수익 |

### 5.2 위험 설명

| template_id | 조건 | 문구 |
|-------------|------|------|
| summary_risk_low | vol < 10% AND mdd > -10% | 낮은 변동성과 제한적인 낙폭 |
| summary_risk_medium | vol < 20% AND mdd > -20% | 중간 수준의 위험 |
| summary_risk_high | else | 높은 변동성 또는 큰 낙폭 |

### 5.3 균형 설명

| template_id | 조건 | 문구 |
|-------------|------|------|
| summary_balance_stable | vol < 10% AND mdd > -10% | 안정성을 중시한 결과 |
| summary_balance_balanced | vol < 20% AND mdd > -20% | 안정성과 변동성 사이의 균형을 추구한 결과 |
| summary_balance_growth | else | 성장 가능성을 추구하되 변동성을 감수한 결과 |

### 5.4 종합 문장

```
지난 {period_years:.1f}년간 이 포트폴리오는 {return_desc}을 기록했습니다.
이 포트폴리오는 {balance_desc}로 해석할 수 있습니다.
(변동성 {vol_pct:.1f}%, 최대 낙폭 {mdd_pct:.1f}%)
```

## 7. 비교 맥락 (Comparison Context) 템플릿

### 7.1 상대 성과 설명

| template_id | 조건 | 문구 |
|-------------|------|------|
| compare_higher | diff > 2%p | {benchmark_name} 대비 연평균 {diff_pct:.1f}%p 높은 수익률을 기록했습니다. |
| compare_lower | diff < -2%p | {benchmark_name} 대비 연평균 {abs_diff_pct:.1f}%p 낮은 수익률을 기록했습니다. |
| compare_similar | -2%p <= diff <= 2%p | {benchmark_name}와 유사한 수익률을 기록했습니다. |
| compare_no_data | benchmark = None | 비교 대상 데이터가 없습니다. |

### 7.2 주의사항 문구

```
비교 수치는 동일 기간 기준입니다.
과거 상대 성과가 미래에도 지속된다는 보장은 없습니다.
```

## 8. 면책 조항 (Disclaimer)

```
본 분석은 과거 데이터에 기반한 정보 제공 목적으로만 작성되었습니다.
과거 성과는 미래 수익을 보장하지 않으며, 투자 권유나 추천이 아닙니다.
모든 투자 결정은 본인의 판단과 책임 하에 이루어져야 합니다.
본 서비스는 투자자문업에 해당하지 않습니다.
```

## 9. Fallback 템플릿

상태 조합이 매핑되지 않거나 예외 발생 시 사용되는 기본 문구입니다.

| template_id | 용도 | 문구 |
|-------------|------|------|
| fallback_cagr | CAGR 기본값 | 해당 기간의 연평균 수익률을 나타냅니다. |
| fallback_vol | 변동성 기본값 | 해당 기간의 연간 변동성을 나타냅니다. |
| fallback_mdd | MDD 기본값 | 해당 기간의 최대 낙폭을 나타냅니다. |
| fallback_sharpe | 샤프 기본값 | 위험 대비 수익을 나타내는 지표입니다. |
| fallback_summary | 요약 기본값 | 해당 포트폴리오의 과거 성과 분석 결과입니다. |
| fallback_neutral | 범용 기본값 | 해당 지표에 대한 정보입니다. |
