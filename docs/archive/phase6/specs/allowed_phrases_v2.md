# Phase 6 허용 문구 v2

목적: 결과 설명 문구가 정보 제공 범위를 넘지 않도록 허용 표현만 고정한다.

## 허용 표현 원칙
- 모든 설명은 과거형으로 표기한다
- “과거 ○○ 기간 기준”이 명시되어야 한다
- “사용자가 직접 선택한 구성”이 명시되어야 한다

## 허용 문구 예시
- “과거 2019-01-01 ~ 2024-12-31 기간의 데이터 기준으로 계산된 결과입니다.”
- “사용자가 직접 선택한 구성에 대해 과거 데이터로 계산된 분석 결과입니다.”
- “해당 기간에 다음 지표가 관측되었습니다.”
- “이 결과는 과거 데이터 기반 통계 정보입니다.”
- “본 결과는 정보 제공 목적이며 투자 자문 또는 권유가 아닙니다.”

## 금지 회피 가이드
- 비교/우열 표현(더 낫다, 우수 등) 포함 금지
- 미래 암시 표현(향후, 기대, 전망 등) 포함 금지
- 권유/추천 표현 포함 금지
