# Phase 6 고지 문구 v2

## 고지 문구 (고정)
본 결과는 사용자가 직접 선택한 구성에 대해 과거 데이터로 계산된 분석 결과입니다.  
본 서비스는 투자 자문 또는 투자 권유를 제공하지 않으며,  
미래 성과를 예측하거나 보장하지 않습니다.
