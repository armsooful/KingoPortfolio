## 1. 금지 표현 목록 (Task 2.1.1)

본 목록은 결과 화면, API 응답, UI 요소에서 **절대 사용 금지**되는 표현이다.

### 1.1 추천·권유 표현

| 분류 | 금지 표현 |
|------|-----------|
| 직접 추천 | 추천, 권유, 제안, 권장, 선택하세요, ~하시기 바랍니다 |
| 간접 추천 | 최적, 최선, 이상적, 적합, 유리, 효율적, 효과적 |
| 비교 우열 | 우수, 우월, 뛰어난, 더 낫다, 승자, 1위, 랭킹, 순위 |
| 점수화 | 점수, 등급, 스코어, 평점, A/B/C 등급 |

### 1.2 미래 예측·보장 표현

| 분류 | 금지 표현 |
|------|-----------|
| 미래 암시 | 향후, 앞으로, 기대, 전망, 예상, 예측, 가능성 |
| 성과 보장 | 보장, 확실, 반드시, 틀림없이, 수익 보장 |
| 목표 설정 | 목표 수익, 기대 수익, 예상 수익률, 목표선 |

### 1.3 판단·해석 표현

| 분류 | 금지 표현 |
|------|-----------|
| 성향 판단 | 안정적, 공격적, 보수적, 위험한, 안전한 |
| 원인 귀속 | 원인, 문제, 책임, ~때문에, ~덕분에 |
| 결론 도출 | 따라서, 결론적으로, 결국, 요약하면 (판단 문맥) |
| 경고 문구 | 위험 경고, 주의 필요, 리스크 주의, 손실 우려 |

### 1.4 비교 관련 금지

| 분류 | 금지 표현 |
|------|-----------|
| 우열 비교 | A가 B보다 낫다, 더 좋은 성과, 우위, 열위 |
| 자동 판단 | 자동 우열 라벨링, 승/패 표시, 추천 마크 |
| 외부 비교 | 시장 대비, 지수 대비, 벤치마크 대비, 타 사용자 대비 |

### 1.5 UI 금지 요소

| 분류 | 금지 요소 |
|------|-----------|
| 시각 강조 | 빨강/초록 색상으로 좋음/나쁨 표시 |
| 아이콘 | 엄지척, 별점, 메달, 트로피, 체크마크(우열 암시) |
| 구간 표시 | 추천 구간, 목표선, 기대선, 이상적 범위 |
| 배지 | "Best", "Top", "추천", "인기" 배지 |

## 2. 허용 표현 목록 (Task 2.1.2)

본 목록은 결과 화면에서 **사용 가능한** 표현이다.

### 2.1 허용 표현 원칙

1. **과거형 표기**: 모든 설명은 과거형으로 작성
2. **기간 명시**: "과거 ○○ 기간 기준" 필수 명시
3. **사용자 선택 강조**: "사용자가 직접 선택한 구성" 명시
4. **중립적 서술**: 사실 나열만, 판단/해석 배제

### 2.2 허용 문구 예시

#### 기본 결과 설명
- "과거 2019-01-01 ~ 2024-12-31 기간의 데이터 기준으로 계산된 결과입니다."
- "사용자가 직접 선택한 구성에 대해 과거 데이터로 계산된 분석 결과입니다."
- "해당 기간에 다음 지표가 관측되었습니다."
- "이 결과는 과거 데이터 기반 통계 정보입니다."

#### 지표 설명
- "해당 기간 연평균 수익률(CAGR)은 X%였습니다."
- "해당 기간 변동성은 X%로 측정되었습니다."
- "해당 기간 최대 낙폭(MDD)은 X%였습니다."
- "해당 기간 누적 수익률은 X%였습니다."

#### 비교 결과 설명
- "포트폴리오 A와 B의 과거 지표를 병렬 비교한 결과입니다."
- "두 포트폴리오의 해당 기간 수치는 다음과 같습니다."

#### 입력 옵션 고지
- "본 결과는 KRW 기준으로 계산되었습니다."
- "본 결과는 USD 기준으로 계산되었습니다."
- "본 결과는 가격 수익(Price Return) 기준으로 계산되었습니다."
- "본 결과는 배당을 포함한 총수익(Total Return) 기준으로 계산되었습니다."

#### 데이터 제한 고지
- "선택한 기간의 과거 데이터가 제한되어 일부 계산이 생략될 수 있습니다."

### 2.3 허용 UI 요소

| 분류 | 허용 요소 |
|------|-----------|
| 차트 | NAV 추이, 누적수익률, MDD 드로다운 |
| 그래프 | 동일 축 병렬 그래프 (비교 시) |
| 테이블 | 수치 병렬 나열 테이블 |
| 색상 | 중립 색상 (회색, 파랑 계열), 데이터 구분용 색상 |

## 3. 결과 화면 하단 고정 문구 (Task 2.1.3)

### 3.1 기본 고정 문구 (필수)

> **본 결과는 과거 데이터를 기반으로 한 정보 제공이며, 투자 권유 또는 보장을 의미하지 않습니다.**

### 3.2 확장 고정 문구 (상세 버전)

> **본 결과는 사용자가 직접 선택한 구성에 대해 과거 데이터로 계산된 분석 결과입니다. 본 서비스는 투자 자문 또는 투자 권유를 제공하지 않으며, 미래 성과를 예측하거나 보장하지 않습니다.**

### 3.3 적용 위치

| 화면 | 적용 문구 | 위치 |
|------|-----------|------|
| 결과 요약 | 기본 고정 문구 | 화면 하단 |
| 상세 결과 | 확장 고정 문구 | 화면 하단 |
| 비교 화면 | 확장 고정 문구 | 화면 하단 |
| 저장/내보내기 | 확장 고정 문구 | 문서 하단 |

## 5. 체크리스트

### 5.1 문구 검증 체크리스트

- [ ] 금지 표현 목록의 단어가 포함되지 않았는가?
- [ ] 과거형으로 표기되었는가?
- [ ] 기간이 명시되었는가?
- [ ] 사용자 선택임이 명시되었는가?
- [ ] 판단/해석 문장이 없는가?

### 5.2 UI 검증 체크리스트

- [ ] 추천 구간/목표선이 없는가?
- [ ] 좋음/나쁨 색상 강조가 없는가?
- [ ] 우열 암시 아이콘/배지가 없는가?
- [ ] 비교 시 수치 병렬 나열만 사용하는가?

### 5.3 비교 기능 체크리스트

- [ ] 동일 사용자 포트폴리오만 비교하는가?
- [ ] 시장/지수 대비 비교가 없는가?
- [ ] 자동 우열 판단이 없는가?
