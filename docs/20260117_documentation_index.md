## 파일명 규칙

```
YYYYMMDD_descriptive_name.md
```

- **YYYYMMDD**: 최초 생성일자 (파일명에 포함)
- **descriptive_name**: 문서 내용을 유추할 수 있는 영문 소문자 이름 (단어는 underscore로 구분)

### 파일 내용 메타데이터

모든 문서 파일은 상단에 다음 순서로 표기합니다:

```markdown
# 문서 제목
최초작성일자: YYYY-MM-DD
최종수정일자: YYYY-MM-DD
```
