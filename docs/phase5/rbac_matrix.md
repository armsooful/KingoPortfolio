# Phase 5 RBAC 매트릭스 (Admin/Operator/Viewer)

## 역할 정의
- Admin: 모든 기능 접근, 설정/배포/권한 변경 가능
- Operator: 운영/모니터링/장애 대응, 배포 승인 불가
- Viewer: 조회 전용, 변경/배포 불가

## 기능별 접근 권한 매핑

| 기능 | Admin | Operator | Viewer |
|------|:-----:|:--------:|:------:|
| 서비스 상태 조회 | ✅ | ✅ | ✅ |
| 로그 조회 | ✅ | ✅ | ✅ |
| 배포 실행/승인 | ✅ | ❌ | ❌ |
| 롤백 실행 | ✅ | ✅ | ❌ |
| 알림 정책 변경 | ✅ | ✅ | ❌ |
| 사용자 데이터 수정 | ✅ | ❌ | ❌ |
| 설정/권한 변경 | ✅ | ❌ | ❌ |

## 감사 로그 적재 항목
- 조회/변경/배포 행위 기록
- 필수 필드: request_id, user_id, action, target, timestamp, result

## 완료 기준
- 무권한 접근 차단 검증
- 로그로 행위 추적 가능
