## 1. 서비스 개요

### 1.1 서비스명
**Kingo Portfolio 성과 해석 서비스**

### 1.2 서비스 목적
- 포트폴리오 성과 지표에 대한 **교육적 해석** 제공
- 투자자의 **의사결정 지원** (투자 권유 아님)
- 과거 성과에 대한 **맥락적 이해** 도모

### 1.3 서비스 범위
| 포함 | 미포함 |
|------|--------|
| 성과 지표 해석 | 종목 추천 |
| 위험 분석 설명 | 매매 시점 조언 |
| 벤치마크 비교 | 투자 판단 유도 |
| PDF 리포트 생성 | 수익률 보장 문구 |

## 3. 기능별 상세 정의

### 3.1 성과 해석 기능

#### 3.1.1 직접 입력 분석
```
POST /api/v1/analysis/explain/direct
```
- **설명**: 사용자가 직접 입력한 성과 지표 해석
- **입력**: CAGR, 변동성, MDD, 샤프비율, 기간
- **출력**: 구조화된 해석 결과 (JSON)
- **제한**: 티어별 월 분석 횟수 적용

#### 3.1.2 포트폴리오 연동 분석
```
POST /api/v1/analysis/explain/portfolio
```
- **설명**: 저장된 포트폴리오 기반 자동 분석
- **입력**: 포트폴리오 ID, 분석 기간
- **출력**: 구조화된 해석 결과 (JSON)
- **제한**: 티어별 월 분석 횟수 적용

### 3.2 PDF 리포트 기능

#### 3.2.1 기본 PDF
```
POST /api/v1/analysis/explain/pdf
```
- **구성**: 1-2페이지
- **내용**: 요약, 지표 해석, 면책 조항
- **제공 티어**: 전체

#### 3.2.2 프리미엄 PDF
```
POST /api/v1/analysis/premium-report/pdf
```
- **구성**: 5-7페이지
- **내용**:
  1. 표지 (제목, 기간, 고지문구)
  2. Executive Summary (요약, 지표 테이블)
  3. 성과 해석 (지표별 상세)
  4. 위험 구간 분석 (MDD 구간, 회복 과정)
  5. 맥락 비교 (벤치마크 대비)
  6. 종합 해석 (포트폴리오 특성)
  7. 참고 및 고지 (면책 조항)
- **제공 티어**: Standard, Premium

### 3.3 히스토리 기능

#### 3.3.1 히스토리 저장
```
POST /api/v1/analysis/history
```
- **설명**: 분석 결과 저장
- **저장 항목**: 입력 지표, 해석 결과, 생성일
- **제한**: 티어별 저장 건수 제한

#### 3.3.2 히스토리 조회
```
GET /api/v1/analysis/history
```
- **설명**: 저장된 분석 내역 조회
- **정렬**: 최신순
- **페이지네이션**: 지원

#### 3.3.3 히스토리 삭제
```
DELETE /api/v1/analysis/history/{history_id}
```
- **설명**: 특정 분석 내역 삭제
- **권한**: 본인 데이터만 삭제 가능

### 3.4 기간별 비교 기능

#### 3.4.1 기간별 비교 분석
```
POST /api/v1/analysis/compare-periods
```
- **설명**: 여러 기간의 성과 지표 비교
- **입력**: 기간 목록 (2-10개)
- **출력**: 기간별 해석 + 비교 분석
- **제공 티어**: Standard (4개), Premium (10개)

## 4. 사용량 제한 정책

### 4.1 Rate Limiting

| 엔드포인트 | Basic | Standard | Premium |
|------------|-------|----------|---------|
| `/explain/direct` | 10/hour | 30/hour | 100/hour |
| `/explain/portfolio` | 10/hour | 30/hour | 100/hour |
| `/explain/pdf` | 5/hour | 20/hour | 50/hour |
| `/premium-report/pdf` | - | 10/hour | 30/hour |
| `/history` | 30/hour | 100/hour | 무제한 |

### 4.2 저장 용량 제한

| 항목 | Basic | Standard | Premium |
|------|-------|----------|---------|
| 히스토리 건수 | 5 | 50 | 무제한 |
| PDF 저장 기간 | 7일 | 30일 | 90일 |

### 4.3 초과 사용 정책
- **Basic**: 제한 초과 시 다음 달까지 대기 또는 업그레이드 안내
- **Standard/Premium**: 추가 사용량 과금 옵션 제공 (별도 협의)

## 5. 규제 준수 사항

### 5.1 투자자문업 규제 회피

본 서비스는 **자본시장법상 투자자문업에 해당하지 않음**을 명확히 합니다.

| 투자자문업 요소 | 본 서비스 대응 |
|-----------------|----------------|
| 특정 종목 추천 | ❌ 미제공 |
| 매매 시점 조언 | ❌ 미제공 |
| 수익률 보장 | ❌ 금지 |
| 개인별 맞춤 조언 | ❌ 일반적 해석만 제공 |

```

투자 권유나 자문이 아닙니다.
과거 성과가 미래 수익을 보장하지 않습니다.
모든 투자 판단의 책임은 본인에게 있습니다.
```
