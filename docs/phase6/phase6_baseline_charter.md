# Phase 6 기준선 선언서 (Scale & Intelligence Baseline Charter)

## 1. 문서 목적
본 문서는 Phase 6(확장·고도화 단계)에서 시스템이 어디까지 허용되고, 어디서부터 금지되는지를 명확히 정의한다.

Phase 6은 기능 확장이 아니라 신뢰·안정·지속성의 고정 단계이며, 본 기준선은 이후 모든 기술·운영 판단의 최상위 기준으로 적용된다.

## 2. Phase 6의 공식 정의
Phase 6이란:

> “추천·권유 없이, 내부 판단 구조를 고도화하되, 외부에서는 정보 제공 서비스의 지위를 유지하는 단계”

즉,

- ❌ 투자 판단을 대신하지 않는다
- ❌ 개인별 맞춤 결론을 제공하지 않는다
- ✅ 판단을 돕는 구조와 근거를 투명하게 제공한다

## 3. 절대 금지 영역 (Red Line)
Phase 6에서는 아래 항목을 기술적으로 가능하더라도 절대 구현하지 않는다.

### 3.1 투자 권유·추천 관련
- 개별 종목 추천
- “매수/매도/보유” 직접적 판단 제시
- 사용자별 최적 포트폴리오 제안
- 성과 기대치(수익률, 목표 금액) 제시

📌 내부 판단 보조 로직이 존재하더라도 외부 노출 금지

### 3.2 AI 표현·포장 관련
- “AI가 추천”
- “AI가 판단”
- “자동으로 최적화된 투자”
- “수익을 극대화하는 알고리즘”

📌 AI/모델은 결정 주체가 아니라 내부 보정 수단으로만 취급

### 3.3 성과 홍보 관련
- 백테스트 수익률 전면 노출
- 실현/미실현 수익 비교 강조
- 타 서비스 대비 성과 우월성 주장

📌 성과는 운영 검증용 내부 지표로만 사용

## 4. 허용 영역 (Allowed Zone)

### 4.1 내부 판단 고도화 (허용)
- Rule 기반 판단 유지
- 시장 신호(변동성, 거래량, 추세 등)를 활용한 가중치 보정
- 전략별 성과 분석 및 내부 피드백 루프

📌 단, 결과는 “정보적 상태”로만 표현

### 4.2 외부 제공 정보 (허용)
- 시장 구조 설명
- 자산군 특성 비교
- 전략 유형 소개 (성장형/방어형 등 일반 분류)
- 과거 데이터 기반 통계 정보

📌 모든 표현은 “판단 자료” 수준에 한정

## 5. Phase 6 AI/모델 사용 내부 정의

| 구분 | 허용 여부 | 설명 |
| --- | --- | --- |
| Rule 엔진 | ✅ | 판단의 기준축 |
| Signal 보정 | ✅ | Rule 가중치 조정 |
| ML/통계 모델 | ✅ | 내부 평가·분석용 |
| 사용자 맞춤 추천 | ❌ | 금지 |
| 자동 투자 결정 | ❌ | 금지 |

AI는 ‘결정자’가 아니라 ‘감시자·보정자’ 역할만 수행

## 6. 운영·리스크 기준선

### 6.1 필수 운영 원칙
- 모든 판단 결과는 추적 가능해야 한다
- “왜 이 상태가 나왔는가” 설명 불가 → 배포 불가
- 자동화 로직에는 항상 수동 차단 스위치 존재

### 6.2 규제 대응 원칙
- 회색지대 접근 금지
- “아직 문제 안 됐다”는 판단 근거로 사용 금지
- 규제 해석은 보수적으로, 기술은 그보다 더 보수적으로

## 7. 외부 커뮤니케이션 기준선

### 7.1 허용 문구 예시
- “시장 데이터를 구조적으로 정리합니다”
- “투자 판단을 돕는 정보 제공 서비스입니다”
- “전략 유형과 시장 흐름을 이해하도록 돕습니다”

### 7.2 금지 문구 예시
- “이 종목이 유망합니다”
- “AI가 골라준 포트폴리오”
- “자동으로 수익을 높입니다”

## 8. 기준선 변경 원칙
Phase 6 기준선은 개발 편의로 변경 불가

변경 시:
1. 문서 개정
2. 리스크 검토
3. 외부 메시지 재검증

→ 3단계 필수

## 9. 선언
본 기준선은 Phase 6 전체 기간 동안 모든 코드, 기획, 운영, 홍보의 상위 규칙으로 적용한다.

> 확장은 하되, 넘지 않는다.  
> 고도화는 하되, 대신 판단하지 않는다.
