# Phase 8-B 입력 유형 분류 (v3)

작성일: 2026-01-21  
범위: Phase 8-B (Input Diversity Expansion)

---

## 1. 목적
입력 다양성 확장을 명확한 카테고리로 고정하고,  
추천/판단 없이 선택 옵션만 제공한다.

---

## 2. 자산군 (Asset Class)
| 코드 | 설명 |
| --- | --- |
| EQUITY | 국내 주식 |
| BOND | 채권 |
| COMMODITY | 원자재 |
| GOLD | 금 |
| REIT | 리츠 |
| ETF | ETF (해외 포함) |

---

## 3. 통화 기준 (Currency)
| 코드 | 설명 |
| --- | --- |
| KRW | 원화 기준 |
| USD | 달러 기준 |

---

## 4. 수익 기준 (Return Type)
| 코드 | 설명 |
| --- | --- |
| PRICE | 가격 수익 기준 |
| TOTAL_RETURN | 배당 포함 총수익 기준 |

---

## 5. 표현 원칙
- 입력은 선택만 제공한다.
- 어떤 옵션이 “좋은지/유리한지” 설명하지 않는다.
- 선택에 따른 차이는 고지 문구로만 알린다.
