# Phase 6 비교 기능 안전 규칙 v2

목적: 비교 기능이 우열 판단이나 추천으로 해석되지 않도록 제한한다.

## 1) 비교 대상 제한
- 동일 사용자 포트폴리오만 비교 가능
- 다른 사용자 포트폴리오 비교 불가
- 시장/지수 대비 비교 기능 제공 금지

## 2) 비교 결과 표현
- 수치 병렬 나열 방식만 허용
- 자동 우열 판단/라벨링 금지
- 색상/아이콘/배지로 우열 암시 금지

## 3) 체크리스트
- [ ] 동일 사용자 포트폴리오만 비교 가능한가?
- [ ] 다른 사용자 포트폴리오 비교 불가한가?
- [ ] 시장/지수 대비 비교 기능이 없는가?
- [ ] 수치 병렬 나열 방식인가?
- [ ] 자동 우열 판단이 없는가?
- [ ] 색상/아이콘으로 우열 암시가 없는가?
