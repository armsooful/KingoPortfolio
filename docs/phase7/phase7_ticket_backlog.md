# Phase 7 티켓 목록

## EPIC P7 — 과거 데이터 기반 포트폴리오 평가 기능 공개

* * *

## 🟦 P7-001 포트폴리오 구성 입력 UI

### 목적

사용자가 **시스템 개입 없이** 종목 또는 섹터를 직접 선택할 수 있게 한다.

### 작업 내용

- 종목 검색/선택 UI
- 섹터 선택 UI(사전 정의 목록)
- 종목/섹터 혼용 불가 처리

### 완료 기준 (AC)

- 추천/기본 선택값 없음
- 시스템 자동 선택 로직 없음
- “추천/우수/최적” 관련 문구 0건

* * *

## 🟦 P7-002 비중 입력 및 검증

### 목적

포트폴리오 비중을 **명시적·결정론적**으로 입력하게 한다.

### 작업 내용

- 비중 입력 필드
- 합계 100% 실시간 검증
- 오류 메시지 중립 처리

### 완료 기준 (AC)

- 합계 ≠ 100% 시 다음 단계 진입 불가
- 오류 문구에 판단/권유 표현 없음

* * *

## 🟦 P7-003 평가 설정 입력

### 목적

과거 평가의 **조건을 명확히 고정**한다.

### 작업 내용

- 분석 기간(시작/종료일)
- 리밸런싱 가정(없음/월/분기)

### 완료 기준 (AC)

- 미래 시점 선택 불가
- 기본값 제공 가능(설명 중립)

* * *

## 🟩 P7-004 과거 가격 데이터 로딩

### 목적

평가에 필요한 **과거 시계열 데이터**를 안정적으로 확보한다.

### 작업 내용

- 종목/섹터별 가격 시계열 조회
- 데이터 출처/기준일 관리

### 완료 기준 (AC)

- 데이터 누락 시 중립 오류 처리
- 오류 메시지에 “다른 선택 권유” 없음

* * *

## 🟩 P7-005 포트폴리오 시계열(NAV) 계산

### 목적

입력된 구성으로 **결정론적 포트폴리오 시계열**을 생성한다.

### 작업 내용

- 비중 기반 NAV 계산
- 리밸런싱 가정 반영

### 완료 기준 (AC)

- 동일 입력 → 동일 결과
- 계산 로직에 최적화/추천 없음

* * *

## 🟩 P7-006 핵심 지표 계산

### 목적

과거 성과·리스크를 **객관 지표로만** 산출한다.

### 작업 내용

- 누적수익률
- CAGR
- 변동성
- 최대 손실폭(MDD)

### 완료 기준 (AC)

- Output Schema v2 100% 준수
- 점수/랭킹/상태 라벨 없음

* * *

## 🟨 P7-007 결과 요약 화면

### 목적

평가 결과를 **정보 형태로만** 전달한다.

### 작업 내용

- 지표 표 형태 출력
- 분석 기간 명시

### 완료 기준 (AC)

- 형용사/판정 문구 없음
- 고지 문구 고정 노출

* * *

## 🟨 P7-008 시계열 차트 화면

### 목적

성과를 **관측 그래프**로만 시각화한다.

### 작업 내용

- NAV 추이
- 누적수익률
- MDD 드로다운

### 완료 기준 (AC)

- 목표선/추천 구간 없음
- 색상으로 우열 암시 없음

* * *

## 🟧 P7-009 포트폴리오 저장

### 목적

사용자가 자신의 구성을 **재사용·비교**할 수 있게 한다.

### 작업 내용

- 포트폴리오 이름/메모 저장
- 구성 + 설정 저장

### 완료 기준 (AC)

- 저장 시 권유/판단 문구 없음
- 동일 구성 재현 가능

* * *

## 🟧 P7-010 평가 결과 히스토리

### 목적

과거 평가 결과를 **그대로 재열람**하게 한다.

### 작업 내용

- 히스토리 목록
- 상세 결과 재조회

### 완료 기준 (AC)

- 재조회 결과 동일
- 결과 수정/재해석 없음

* * *

## 🟥 P7-011 자기 포트폴리오 비교 대상 선택

### 목적

비교 범위를 **“나 vs 나”로 제한**한다.

### 작업 내용

- 동일 사용자 포트폴리오 선택
- 선택 개수 제한

### 완료 기준 (AC)

- 타 사용자 접근 불가
- 서버 단 검증 포함

* * *

## 🟥 P7-012 비교 결과 화면

### 목적

비교를 제공하되 **결론은 사용자에게 남긴다**.

### 작업 내용

- 수치 병렬 비교
- 동일 축 차트 병렬

### 완료 기준 (AC)

- 승자/우열/점수 표시 없음
- comparison\_rules\_v2.md 준수

* * *

## 🟪 P7-013 Phase 6 금지어 자동 검증 연계

### 목적

표현 리스크를 **코드 레벨에서 차단**한다.

### 작업 내용

- 출력 텍스트 금지어 스캔
- CI 파이프라인 연동

### 완료 기준 (AC)

- 금지어 1건이라도 있으면 빌드 실패

* * *

## 🟪 P7-014 Golden Test 통합

### 목적

Phase 6 경계를 **기계적으로 보증**한다.

### 작업 내용

- Golden Output 비교
- 비교 화면 테스트 포함

### 완료 기준 (AC)

- Golden Test 100% 통과
- 실패 시 배포 차단
