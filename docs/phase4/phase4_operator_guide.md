# Phase 4 운영자 가이드

## 목적
신규 운영자가 단독으로 운영/대응할 수 있도록 절차를 제공한다.

## 일일 운영 체크
- 서비스 상태 확인(`/health`)
- 알림 채널 확인
- 에러 로그 요약 점검

## 배포 시 체크
- 배포 체크리스트 실행
- 핵심 플로우 스모크 테스트
- SLA 지표 확인

## 장애 대응 흐름
- Runbook 시나리오 매칭
- 임시 조치 및 롤백 판단
- 사후 보고 작성
