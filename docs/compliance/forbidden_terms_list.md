# 금지어 목록 (Forbidden Terms)
최초작성일자: 2026-01-14
최종수정일자: 2026-01-20

본 문서는 Foresto Compass 서비스에서 사용이 금지된 용어를 정의합니다.
한국 자본시장법 및 금융소비자보호법에 따라 투자일임업·투자자문업 미등록 서비스에서 사용할 수 없는 표현입니다.

## 목적

- PR 머지 전 자동 검사를 통해 금지어 유입 방지
- 규제 준수 및 법적 리스크 최소화
- 일관된 교육 목적 서비스 용어 사용

## 금지어 목록

### 1. 추천/권유 관련 (HIGH - 즉시 수정 필요)
| 금지어 | 대체 표현 | 사유 |
|--------|-----------|------|
| 추천 | 예시, 학습용 시나리오 | 투자 권유로 오인 가능 |
| 권유 | 안내, 설명 | 투자 권유로 오인 가능 |
| 매수/매도 추천 | 시뮬레이션 예시 | 직접적 투자 지시 |
| 투자하세요 | 학습해보세요 | 행동 유도 표현 |
| 사세요/파세요 | - | 직접적 거래 지시 |

### 2. 수익 보장 관련 (HIGH - 즉시 수정 필요)
| 금지어 | 대체 표현 | 사유 |
|--------|-----------|------|
| 보장 | 과거 실적 기반 참고치 | 미래 수익 보장 오해 |
| 확실한 수익 | 과거 평균 수익률 | 확정 수익 오해 |
| 손실 없음 | - | 허위 정보 |
| 무위험 | 저위험 (참고용) | 과장 표현 |
| 안전한 투자 | 안정 추구형 시나리오 | 원금 보장 오해 |

### 3. 예측/전망 관련 (MEDIUM - 수정 권장)
| 금지어 | 대체 표현 | 사유 |
|--------|-----------|------|
| 기대수익률 | 과거 평균 수익률 (참고용) | 미래 예측으로 오해 |
| 예상수익 | 과거 실적 기반 시뮬레이션 | 미래 예측으로 오해 |
| 기대 | 과거 관측 결과 | 미래 예측으로 오해 |
| 전망 | 과거 추세 분석 | 예측 서비스 오해 |
| 향후 | 과거 기간 기준 | 미래 예측으로 오해 |
| 가능성 | 과거 관측 결과 | 미래 예측으로 오해 |
| ~할 것입니다 (수익 관련) | ~했습니다 (과거형) | 미래 보장 표현 |

### 4. 맞춤/개인화 관련 (MEDIUM - 수정 권장)
| 금지어 | 대체 표현 | 사유 |
|--------|-----------|------|
| 맞춤형 포트폴리오 | 학습 시나리오 | 자문 서비스 오해 |
| 개인 맞춤 추천 | 유형별 예시 | 자문 서비스 오해 |
| 당신을 위한 | 선택한 시나리오의 | 개인 타겟팅 오해 |

### 6. 평가/비교/우열 관련 (MEDIUM - 수정 권장)
| 금지어 | 대체 표현 | 사유 |
|--------|-----------|------|
| 최적 | 비교 대상 없음 | 우열 판단 오해 |
| 우수 | 관측된 지표 | 성과 우열 오해 |
| 효율 | 계산된 수치 | 성과 우열 오해 |
| 승자 | 비교 미제공 | 승패 판단 오해 |
| Top | 상위 표기 금지 | 순위화 오해 |
| 랭킹 | 순서 표기 금지 | 순위화 오해 |
| 더 낫다 | 차이 있음 | 우열 판단 오해 |
| 불리하다 | 차이 있음 | 우열 판단 오해 |
| 안정적 | 변동성 낮음 | 성향 판단 오해 |
| 공격적 | 변동성 높음 | 성향 판단 오해 |

### 5. 전문가/자문 관련 (HIGH - 즉시 수정 필요)
| 금지어 | 대체 표현 | 사유 |
|--------|-----------|------|
| 전문가 조언 | 일반 정보 | 자문 서비스 오해 |
| 투자 자문 | 학습 도구 | 미등록 자문 |
| 투자 상담 | 용어 설명 | 미등록 상담 |

## 예외 사항

다음 컨텍스트에서는 금지어 사용이 허용됩니다:

1. **면책 조항 (Disclaimer)**: "본 서비스는 투자 추천이 아닙니다" 등
2. **부정문**: "추천이 아닙니다", "보장하지 않습니다" 등
3. **테스트 코드**: 단위 테스트, E2E 테스트 파일
4. **문서**: 금지어 목록 문서 자체, 변경 로그

## 스크립트 사용법

```bash
# 금지어 검사 실행
./scripts/forbidden_terms_check.sh

# 특정 디렉토리만 검사
./scripts/forbidden_terms_check.sh frontend/src

# CI에서 사용
./scripts/forbidden_terms_check.sh && echo "통과" || exit 1
```
