# KingoPortfolio 용어 정리 가이드
최초작성일자: 2026-01-05
최종수정일자: 2026-01-18

본 문서는 KingoPortfolio 서비스 전반에서 사용되는 핵심 용어를 정리하고 표준화합니다.

## 1. 투자 성향 관련 용어

### 투자 성향 (Investment Type)
- **표준 용어**: "투자 성향"
- **영문**: Investment Type
- **사용 예시**: "귀하의 투자 성향을 분석합니다"
- **금지 용어**: 투자성향 (띄어쓰기 누락)

### 투자 성향 분류
1. **안정형** (Conservative)
   - 설명: 원금 보존을 최우선으로 하는 보수적 투자 성향
   - 특징: 낮은 위험, 안정적 수익 추구

2. **중립형** (Moderate)
   - 설명: 안정성과 수익성의 균형을 추구하는 투자 성향
   - 특징: 중간 수준의 위험 감수

3. **공격형** (Aggressive)
   - 설명: 높은 수익을 위해 높은 위험을 감수하는 투자 성향
   - 특징: 높은 위험, 높은 기대 수익

## 2. 리스크 및 위험도 용어

### 리스크 레벨 (Risk Level)
- **표준 용어**: "리스크 레벨" 또는 "위험 수준"
- **영문**: Risk Level
- **사용 예시**: "이 상품의 리스크 레벨은 중간입니다"
- **금지 용어**: "위험도" (모호함)

### 위험 감내도 (Risk Tolerance)
- **표준 용어**: "위험 감내도"
- **영문**: Risk Tolerance
- **설명**: 투자자가 받아들일 수 있는 위험의 정도
- **사용 예시**: "귀하의 위험 감내도를 평가합니다"

## 3. 수익률 관련 용어

### 기대 수익률 (Expected Return)
- **표준 용어**: "기대 수익률" (연간 기준 명시 필요시 "연간 기대 수익률")
- **영문**: Expected Annual Return
- **표기 방식**: 퍼센트 표기 시 소수점 둘째 자리까지 (예: 7.50%)
- **사용 예시**: "연간 기대 수익률: 7.50%"
- **금지 용어**: "예상 수익률", "기대수익률" (띄어쓰기 누락)

### 실현 수익률 (Actual Return)
- **표준 용어**: "실현 수익률" 또는 "실제 수익률"
- **영문**: Actual Return
- **설명**: 과거에 실제로 달성한 수익률
- **사용 예시**: "지난 1년간 실현 수익률: 12.35%"

### 총 수익률 (Total Return)
- **표준 용어**: "총 수익률"
- **영문**: Total Return
- **설명**: 자본 이득과 배당을 포함한 전체 수익률
- **사용 예시**: "총 수익률: 15.20%"

### 기간 수익률 (Period Return)
- **표준 용어**: "기간 수익률" (기간 명시 필요)
- **영문**: Period Return
- **사용 예시**: "90일 기간 수익률: 5.40%"

## 4. 포트폴리오 관련 용어

### 포트폴리오 (Portfolio)
- **표준 용어**: "포트폴리오"
- **영문**: Portfolio
- **설명**: 투자자가 보유한 자산의 조합
- **사용 예시**: "맞춤 포트폴리오를 생성합니다"

### 자산 배분 (Asset Allocation)
- **표준 용어**: "자산 배분"
- **영문**: Asset Allocation
- **설명**: 주식, ETF, 채권, 예금 등 자산 유형별 투자 비중
- **사용 예시**: "자산 배분: 주식 60%, 채권 30%, 예금 10%"

### 포트폴리오 구성 (Portfolio Composition)
- **표준 용어**: "포트폴리오 구성"
- **영문**: Portfolio Composition
- **설명**: 포트폴리오 내 개별 종목의 구성 내역
- **사용 예시**: "포트폴리오 구성을 확인하세요"

## 5. 종목 및 상품 용어

### 종목 코드 (Ticker Symbol)
- **표준 용어**: "종목 코드"
- **영문**: Ticker / Ticker Symbol
- **사용 예시**: "종목 코드: 005930"
- **금지 용어**: "티커", "심볼"

### 시가총액 (Market Capitalization)
- **표준 용어**: "시가총액"
- **영문**: Market Capitalization
- **단위**: 억원 (한국 시장 기준)
- **사용 예시**: "시가총액: 8,591,617억원"

### 재무 지표 (Financial Indicators)
- **PER** (Price-to-Earnings Ratio): 주가수익비율
- **PBR** (Price-to-Book Ratio): 주가순자산비율
- **ROE** (Return on Equity): 자기자본이익률
- **배당수익률** (Dividend Yield): 배당금/주가 비율

## 6. 시계열 및 차트 용어

### 시계열 데이터 (Time Series Data)
- **표준 용어**: "시계열 데이터"
- **영문**: Time Series Data
- **설명**: 시간 순서대로 나열된 데이터
- **사용 예시**: "90일 시계열 데이터를 조회합니다"

### 가격 차트 (Price Chart)
- **표준 용어**: "가격 차트"
- **영문**: Price Chart
- **포함 정보**: 시가, 종가, 고가, 저가
- **사용 예시**: "가격 차트를 확인하세요"

### 거래량 차트 (Volume Chart)
- **표준 용어**: "거래량 차트"
- **영문**: Volume Chart
- **사용 예시**: "거래량 차트를 표시합니다"

## 7. 백테스팅 용어

### 백테스트 / 백테스팅 (Backtesting)
- **표준 용어**: "백테스팅" (동사형) / "백테스트" (명사형)
- **영문**: Backtesting
- **설명**: 과거 데이터로 투자 전략을 검증하는 과정
- **사용 예시**:
  - "포트폴리오 백테스팅 결과"
  - "백테스트를 실행합니다"

### 시뮬레이션 (Simulation)
- **표준 용어**: "시뮬레이션"
- **영문**: Simulation
- **설명**: 가상의 투자 상황을 모의 실험
- **사용 예시**: "1년 기간 시뮬레이션 결과"

## 8. 금액 표기 기준

### 한국 원화 표기
- **기본 단위**: 원 (KRW)
- **천 단위 구분**: 쉼표 사용 (예: 10,000,000원)
- **대규모 금액**:
  - 만원 단위: "1,000만원"
  - 억원 단위: "100억원"

### 퍼센트 표기
- **소수점**: 기본 소수점 둘째 자리까지 (예: 12.34%)
- **음수**: 마이너스 기호 사용 (예: -5.20%)
- **소수점 생략 가능**: 정수인 경우 (예: 10%)

## 9. 날짜 및 기간 표기

### 날짜 표기
- **표준 형식**: YYYY-MM-DD (예: 2026-01-05)
- **한글 표기**: YYYY년 MM월 DD일 (예: 2026년 1월 5일)

### 기간 표기
- **일 단위**: "90일", "30일 기간"
- **월 단위**: "3개월", "6개월"
- **년 단위**: "1년", "3년"

## 12. 면책 문구 표준

모든 투자 정보 제공 페이지에는 적절한 면책 문구를 포함해야 합니다:

- **일반**: 투자 참고용, 권유 아님
- **진단**: 참고용 정보, 금융기관 정식 확인 필요
- **포트폴리오**: 알고리즘 기반 참고 자료
- **종목 정보**: 참고용, 실시간 아닐 수 있음
- **백테스트**: 과거 시뮬레이션, 실제 결과와 다를 수 있음

2. **중간**: API 응답 메시지, 에러 메시지
