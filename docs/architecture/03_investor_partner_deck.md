## 1. Executive Summary

### ForestoCompass는?

**교육 목적의 포트폴리오 시뮬레이션 플랫폼**

사용자가 다양한 투자 시나리오를 과거 데이터로 시뮬레이션하고,
그 결과를 정량적 지표로 이해할 수 있는 교육 도구입니다.

```
┌─────────────────────────────────────────────┐
│              핵심 가치 제안                  │
├─────────────────────────────────────────────┤
│                                             │
│  "투자 전에 시뮬레이션으로 학습하세요"        │
│                                             │
│  • 과거 데이터 기반 시나리오 분석             │
│  • 위험 지표(MDD, 변동성) 직관적 이해         │
│  • 리밸런싱 효과 시뮬레이션                  │
│                                             │
└─────────────────────────────────────────────┘
```

## 4. 기술 역량

### 4.1 아키텍처

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│   Frontend   │     │   Backend    │     │   Database   │
│   React/TS   │ ◄─► │   FastAPI    │ ◄─► │  PostgreSQL  │
└──────────────┘     └──────────────┘     └──────────────┘
       │                    │
       ▼                    ▼
┌──────────────┐     ┌──────────────┐
│   Vercel     │     │   Railway    │
│   (CDN)      │     │   (Server)   │
└──────────────┘     └──────────────┘
```

### 4.2 기술 스택

| 영역 | 기술 | 선택 이유 |
|------|------|-----------|
| Frontend | React + TypeScript | 타입 안정성, 생태계 |
| Backend | FastAPI (Python) | 금융 라이브러리 호환 |
| Database | PostgreSQL | JSONB, 확장성 |
| Infra | Vercel + Railway | 빠른 배포, 확장성 |

### 4.3 품질 지표

| 지표 | 현재 |
|------|------|
| 단위 테스트 | 57개 (Phase 2) |
| 테스트 커버리지 | 핵심 로직 100% |
| API 응답 시간 | < 500ms (캐시 히트) |
| 가동률 목표 | 99.9% |
