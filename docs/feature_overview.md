# 기능 설명서 (ForestoCompass)
최초작성일자: 2026-01-20
최종수정일자: 2026-01-20

## 개요
ForestoCompass는 투자 권유·추천·자문을 제공하지 않는 정보 제공 플랫폼이다.  
사용자가 선택한 구성에 대해 과거 데이터 기반의 분석/비교 결과를 제공하고, 운영·감사 추적을 위한 로그 체계를 포함한다.

## 기능 목록

### 사용자 기능
- 회원가입/로그인/프로필 조회·수정
- 설문 기반 진단 제출 및 진단 결과 조회
- 포트폴리오 생성/조회/비교(정보 제공 목적)
- 북마크 저장 및 사용자 설정(프리셋/알림 등)
- 결과 리포트(PDF) 생성

### 데이터/분석 기능
- 시장 데이터 조회 및 시계열 제공
- 시뮬레이션/백테스트 실행
- 성과 분석(과거 지표 중심)
- 결과 비교(동일 사용자 범위 내)
 - 외부 API 연동 데이터 수집(시장/가격/지표)

### 운영/관리 기능
- 관리자 제어(승인/정정/배포 승인)
- 관리자 모드(권한 기반 접근, 관리 화면/엔드포인트)
- 데이터 품질/계보 점검
- 배치 작업 관리 및 실행 이력 추적
- 감사 로그/운영 알림 기록
- Rate Limit/가드 규칙 적용

### 규제/안전 기능
- 금지어/표현 가드(추천·우열·예측 표현 차단)
- 고지 문구 적용(정보 제공 목적)
- 권한 분리(RBAC) 및 무권한 차단

## 주요 플로우

1) 사용자 온보딩
- 회원가입 → 로그인 → 설문 진단 제출 → 진단 결과 확인

2) 포트폴리오 분석
- 사용자 구성 선택 → 포트폴리오 생성 → 성과/리스크 지표 확인 → 비교(동일 사용자 범위)

3) 리포트 출력
- 분석 결과 확인 → PDF 리포트 생성 → 저장/내보내기

4) 운영/감사
- 관리자 승인/정정 → 감사 로그 적재 → 알림 기준 충족 시 운영 알림

## 제한사항
- 개별 종목 추천/매수·매도 판단 제공 없음
- 사용자별 최적 포트폴리오 제안 없음
- 미래 성과 예측/보장 표현 금지
- 타 사용자/시장 대비 우열 비교 금지
